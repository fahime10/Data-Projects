# Global Cybersecurity Threats (2015-2024) EDA
This project showcases data analytical skills.

This project analyzes global security incidents between 2015 and 2024 to identify trends in attack types, financial losses, affected industries, and incident resolution times. The analysis combines Python for statistical analysis and Power BI for interactive data visualization.

## Key business questions
* Which countries experienced the highest growth in affected users from cyberattacks between 2015 and 2024?
* Which industries suffered the most financially and affected the users?
* Which attack types caused the most financial losses?
* Are resolution times improving?

## Dataset
Dataset source: Kaggle  
https://www.kaggle.com/datasets/atharvasoundankar/global-cybersecurity-threats-2015-2024/data  
The dataset contains 3,000 rows, with a time range of 2015-2024.

## Tools
Python (pandas, statsmodels, matplotlib);  
Microsoft Power BI

## Project workflow
1. Data cleaning and preprocessing using Python
2. Exploratory Data Analysis
3. Statistical analysis using OLS regression
4. Creation of interactive dashboards in Power BI
5. Interpretation of results and report writing

## Key findings
* DDoS attacks caused the highest financial losses between 2015-2024
* IT, banking, and government industries experienced the greatest impact, both financially and user-wise
* Several countries showed increase in affected users, although trends were not statistically significant
* Resolution times did not significantly improve over the observed period

## Project structure

## Limitations
The dataset contains aggregated values, which required some business questions to be rephrased.
